# InviteMe Mobile

React Native mobile application for InviteMe using Expo.

## Development

```bash
npm run dev
```

## Building

```bash
# Android
npm run android

# iOS
npm run ios

# Web
npm run web
```

## Prerequisites

- Node.js >= 18
- Expo CLI (installed globally or via npx)
- For iOS: Xcode and iOS Simulator
- For Android: Android Studio and Android Emulator
